## Tecnologias
- Node.js
- MySQL (XAMPP)
- Prisma
- React.js
- Vite

## Tutorial para executar API 

- Entre na pasta `./api` e cole os seguintes comandos.

```bash
npm install
npx prisma migrate dev --name init
npx nodemon
```

Agora voce pode testar o Back-End
