## Tutorial para executar Web

- Clone o repositorio e apos clonar entre na pasta `./web` e cole o seguintes comandos.

```bash
cd web
npm install
npm run dev
```

Apos isso clique no link `http://localhost:5173/`.

Agora voce pode testar o Front-End
